Me, Cody Holland, Ethan Sanford, and Atmesh Acharya worked on a Spotify Integration with games for our Senior Capstone Project:

Using Spotify's Web API, we managed to make song requests in the user's behalf.
The goal was to help game developers by passing the cost to the user when it came to the game's soundtrack.

It was a long project, so I love looking back and comparing the end result with what we had 1 week in.
We planned on making a demo game to showcase song requests. That would require recognizing game events, like collisions.
The first thing we had to show for was a small box game written in JavaScript that recognizes collisions.
